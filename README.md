# 🚀 Smart EventFlow AI

An AI-powered real-time stadium assistant designed to improve crowd management, navigation, and user experience in large-scale events.

---

## 🎯 Problem Statement

Managing crowd movement in stadiums is complex due to:
- High congestion zones
- Long waiting times
- Lack of real-time guidance

---

## 💡 Solution

Smart EventFlow AI provides:
- 🔥 Live crowd heatmap visualization
- 📍 Smart zone-based navigation
- 🤖 AI chatbot for assistance
- ⚡ Real-time updates using Firebase

---

## ⚙️ Tech Stack

- HTML, CSS, JavaScript
- Firebase Realtime Database
- Firebase Hosting

---

## ☁️ Google Services Used

- Firebase Realtime Database → real-time crowd sync
- Firebase Hosting → live deployment

Future Scope:
- Google Maps API for navigation
- Gemini AI for advanced assistant

---

## 🧠 How It Works

1. Simulates crowd data
2. Stores data in Firebase
3. Syncs data in real-time
4. Updates UI dynamically
5. Provides AI-based suggestions

---
